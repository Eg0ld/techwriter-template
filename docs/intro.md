# 🪄 Your documentation is being generated

This site was created by **techwriter.ai**. Right now, the **Autoscribe** engine
is reading your repository and writing the first set of pages — it runs as a
GitHub Action and usually takes a few minutes.

:::info Build in progress — nothing is broken
This placeholder is replaced automatically once your documentation is generated
and merged. Check back shortly.
:::

## What happens next

1. ✅ Your docs repository was created.
2. ⏳ **Autoscribe generates the initial documentation** from your code.
3. 📝 It opens a **draft pull request** with the generated docs.
4. 🚀 **Merge that PR** — this page is replaced by your real documentation and the
   site redeploys automatically.

## Track progress

- **Actions** tab → the _Autoscribe · Initial docs_ run shows live progress.
- **Pull requests** → your generated docs arrive as a draft PR to review and merge.

## It's your repo

You don't have to wait for the engine — this is an ordinary Docusaurus site. Add
a `.md`/`.mdx` file under `docs/` and it shows up in the sidebar automatically.
Run it locally with `npm install && npm start`.
