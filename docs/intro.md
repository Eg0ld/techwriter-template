# 🪄 Your documentation is being generated

This site was just created by **techwriter.ai**. Right now its engine is reading
your repository and writing the first set of pages — it runs as a GitHub Action
and usually finishes within a few minutes.

:::info[Build in progress — nothing here is broken]
This is a temporary placeholder. It's replaced automatically the moment your
generated documentation is merged. Check back shortly.
:::

## What happens next

1. ✅ Your docs repository was created.
2. ⏳ The engine **generates your initial documentation** from your code.
3. 🚀 It **publishes automatically** — this page is replaced by your real
   documentation and the site redeploys, no action needed from you.

_After this first build, ongoing updates (as the watched repo changes) arrive as
**draft pull requests** you review and merge — so you stay in control of changes._

## Track progress

- The **Actions** tab → the _Autoscribe · Initial docs_ run shows live progress.
- This page refreshes to your real docs as soon as that run finishes.

## It's your repo

You don't have to wait for the engine — this is an ordinary Docusaurus site. Drop
a `.md` or `.mdx` file under `docs/` and it appears in the sidebar automatically.
Run it locally any time with `npm install && npm start`.
